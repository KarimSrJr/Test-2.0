About Football Hub

A polished, static football website with modern UI — no frameworks required.

What’s new in v2

  • Modern typography with Inter, cleaner spacing, refined colors.
  • Hero media card & KPI tiles.
  • Improved cards, badges, and table styles.
  • Drop-in file structure for GitHub Pages.

Customize

  1. Edit assets/data.json with your teams, fixtures, and scorers.
  2. Tweak colors in assets/styles.css under :root.
  3. Add pages by duplicating an HTML and updating the nav.
